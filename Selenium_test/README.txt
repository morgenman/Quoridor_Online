Two Files in this folder(SeleniumTest.side and test_test01.py) are to be used with Selenium IDE extension In Google Chrome.

------------To Use----------------
Laugh the IDE From Google chrome extension tab then a prompt will open asking you to either open a project or create one, choose open then click on the SeleniumTest folder, then choose the SeleniumTest.side project. After opening the project click on the play button to run all the current test on the webpage. 

The other file is the code of all the test written in python, which includes the WebDriver application in selenium.
